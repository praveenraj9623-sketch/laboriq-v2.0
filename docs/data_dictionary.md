# Data Dictionary

| Column | Meaning |
|---|---|
| job_id | Unique posting identifier |
| posted_date | Date job was posted |
| company | Hiring company |
| job_title | Raw/cleaned job title |
| location | Job location |
| employment_type | Full-time, contract, internship, etc. |
| experience_level | Entry, Associate, Mid, Senior |
| salary_min_lpa | Minimum annual salary in LPA |
| salary_max_lpa | Maximum annual salary in LPA |
| salary_mid_lpa | Mean of min and max salary |
| job_description | Cleaned job description text |
| extracted_skills | Skills extracted from title + description |
| skill_count | Number of extracted skills |
| role_label | Standardized job role label |
| occupation_family | Higher-level occupation group |
| occupation_confidence | Confidence score from mapper |
